# 网站建设变更日志

本文件用于记录 `xosite` 在建设过程中的重要变动。

记录原则：
- 记录“对网站建设有意义的变化”
- 优先写清楚做了什么、为什么改、影响了什么
- 不必记录过碎的小试错

建议记录范围：
- 页面结构新增或调整
- 视觉风格变更
- 技术方案调整
- 部署链路变化
- GitHub / Vercel 相关接入
- 内容模块新增或下线
