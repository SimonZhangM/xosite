# 网站建设变更日志

本文件用于记录 `xosite` 在建设过程中的重要变动。

记录原则：
- 记录“对网站建设有意义的变化”
- 优先写清楚做了什么、为什么改、影响了什么
- 不必记录过碎的小试错

建议记录范围：
- 页面结构新增或调整
- 视觉风格变更
- 技术方案调整
- 部署链路变化
- GitHub / Vercel 相关接入
- 内容模块新增或下线

## [0.1.3] - 2026-04-18

### 首页结构升级

- 重写首页 Hero、问题卡、产品定位、核心能力、界面预览、信任边界与适用人群模块，首页表达从“功能罗列”进一步转向“用户问题与工作台价值”。

### 资源与素材

- 新增一组 SVG 图标资源，用于替换首页多个区块中的占位图形。

### 文档整理

- 更新 `WEBSITE_BUILD_PROGRESS.md`，将其明确作为网站当前进度说明文档使用。
- 更新仓库首页说明与版本描述，使项目介绍与当前网站状态保持一致。

## [0.1.2] - 2026-04-18

### 界面与体验打磨

- 调整首页 Hero、导航与章节节奏，继续收口首屏层级与视觉密度。
- 统一页面背景来源，减少 section 之间因背景叠加产生的横向接缝。

### 资源与品牌

- 新增 `logo.png` 与 `app/favicon.ico`，补充站点品牌资源并替换默认图标。

### 工程整理

- 清理默认示例静态资源文件，保持 `public/` 目录更贴近正式官网用途。
- 同步项目包版本到 `0.1.2`，为后续发布与页头版本联动做准备。
