# xosite

Official website project for XplorOne, built with Next.js.

## What's inside

- Landing page
- Download status page
- FAQ page
- Changelog page
- Product screenshots and static assets

## Local development

```bash
npm install
npm run dev
```

## Tech stack

- Next.js 16
- React 19
- TypeScript

## Notes

- Core site content lives in `app/`
- Static assets and screenshots live in `public/`
- Several project notes and planning documents are kept in the repository root
